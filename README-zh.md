# Yeats Liao 的个人主页与博客

[English](./README.md)

欢迎来到我的个人学术主页与博客！本网站基于 Jekyll 构建，提供了一个现代、响应式的布局，用于展示我的学术论文、项目以及个人博客文章。

## 主要功能

- **关于我**：展示我的教育背景、研究兴趣以及精选的学术论文列表。
- **博客**：采用清晰的时间轴布局，用于分享我的思考、研究进展和技术教程。
- **谷歌学术爬虫**：通过 GitHub Actions 自动获取并更新我的最新谷歌学术发表数据。
- **响应式设计**：完美适配桌面端、平板和移动端设备。

## 本地运行

要在本地运行此网站，请确保已安装 Ruby 和 Bundler。

1. 克隆仓库：
   ```bash
   git clone https://github.com/yeatsliao/yeatsliao.github.io.git
   cd yeatsliao.github.io
   ```

2. 安装依赖：
   ```bash
   bundle install
   ```

3. 启动本地服务器：
   ```bash
   bundle exec jekyll serve
   ```

4. 在浏览器中访问 `http://localhost:4000`。
